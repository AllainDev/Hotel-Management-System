# Hotel Management System

Hotel Management System (HMS) can help you streamline and automate various aspects of managing a hotel. It can help you keep track of room reservations, bills, guest information, and financial transactions. 

## Authors

- [@AllainDev](https://github.com/AllainDev)

## Deployment

Make sure that you have Microsoft Sql Server installed in your PC. Along with that I have already uploaded the script file which you just have to run to create database. Connect your application from Visual Studio with Microsoft Sql Server services of integrating database. You are now good to go!

## Tech Stack

Apply .NET Generic Host & Dependency Injection (DI)

Using 3-Layers architecture to develop the application.

Apply MVVM (Model-View ViewModel) pattern in the application.

Apply Repository pattern and Singleton pattern in a project.

Add CRUD and searching actions to the application.

Apply to validate data type for all fields.
